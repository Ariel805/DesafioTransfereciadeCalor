Cómo funciona?

El usuario ingresa los cuatro valores numéricos en el formulario
Al presionar Calcular Temperatura, el script captura los datos con document.getElementById().value
Se validan los datos (campos vacíos, k > 0, t ≥ 0)
Se aplica la fórmula usando Math.exp() para la parte exponencial
El resultado se redondea al entero más cercano con Math.round()
Se muestra el resultado en pantalla de forma dinámica
